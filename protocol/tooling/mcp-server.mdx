A Model Context Protocol server for YieldPoint is in development. It will expose protocol state and transaction-building tools so AI agents can read vault data and prepare deposit, redeem, withdraw, claim, and bridge calls through a standard MCP interface.

Until it ships, the [API](https://api.yieldpoint.io/docs/json) exposes read-only protocol state, and the [Integration](/protocol/integration/overview) section documents the contract surface for direct on-chain interaction.
